使用 vscode 编写文档时不能实时预览自己的文件样式，帮你解决

# 一、安装插件

```
vscodo 打开插件安装输入 markdown all in one
```

![](/images/vscode-1.png)

# 二、打开 Markdown 文件实现预览

```
- 通过ctrl+shift+p打开vscode配置
- 输入>Markdown: Open Preview to the side 即可打开预览效果图
```

![](/images/vscode-2.png)
